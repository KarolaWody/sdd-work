# Tools for SpecDD with OpenCode

## Required
- **OpenCode CLI** — run the skill (`opencode run skill/sdd_work.md`)
- **Python 3** — runtime for helper scripts
- **Git** — version control for specs and code

## Recommended
- **VSCode** with extensions:
  - **Markdown Preview** — preview spec files
  - **GitLens** — visualize spec and code history
  - **YAML** (by Red Hat) — if using YAML config files
- **OpenCode VSCode extension** — run OpenCode inside the editor

## Optional
- **ruff** — Python linter/formatter (if writing Python helpers)
- **pytest** — test runner (for validation step)
- **Any VSCode theme** — personal preference

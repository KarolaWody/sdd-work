# sdd-work — Usage Examples

## 1. Start a new project with SDD

```
Use skill sdd-work. I want to start a new project.
```

Agent asks 8 onboarding questions, then creates constitution files.

## 2. Plan the first feature

```
Use skill sdd-work. Let's work on phase 1 from the roadmap.
```

Agent creates feature branch and spec files, then interviews user.

## 3. Implement a feature

```
Use skill sdd-work. Implement the feature per the plan.
```

Agent implements task groups and presents changes for review.

## 4. Validate and merge

```
Use skill sdd-work. Run validation.
```

Agent runs @code_checker, shows results, waits for approval.

## 5. Replan

```
Use skill sdd-work. Let's review the roadmap.
```

Agent proposes updates and waits for user approval.

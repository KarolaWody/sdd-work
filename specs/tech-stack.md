# Tech Stack

| Layer | Technology | Cost | Notes |
|---|---|---|---|
| Language | Python 3 | Free | Cross-platform, widely adopted |
| CLI | OpenCode | Free | AI-powered development assistant |
| AI review | OpenCode `@code_checker` sub-agent (North Mini Code) | Free | Runs locally via OpenCode |
| Version control | Git + GitHub | Free | Public repo, portfolio-ready |
| CI/CD | GitHub Actions | Free (2000 min/month) | Future — for automated testing |
| Docs hosting | GitHub Pages | Free | Future — for project site/marketing |
| Development | Any code editor (VSCode recommended) | Free | |

## Constraints
- Zero budget — all tools must be free or have a free tier
- Must work offline-capable where possible (OpenCode + `@code_checker` runs locally)
- Must be reproducible by any OpenCode user

## Conventions
- Python 3.10+ compatible
- English for all code, docs, and comments
